```

3. 金融/货币/债券类      → finance.png    ✨ TASK 6C 新增
```

- 强语义类优先于泛词类

- "城市/生活"优先级低于"金融/科技/安全"等强语义词

## 11. 优化细节

### 11.1 去掉过于通用的词汇

TECH_KEYWORDS 中去掉了 "系统"，原因：
- 过于通用，容易产生误识别
- 例："社区门禁系统升级"中的"系统"不是科技语义
- 保留了更具体的词汇：AI、人工智能、软件、模型等
